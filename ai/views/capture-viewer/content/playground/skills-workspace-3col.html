How do I add a new skill that works in both Cursor and Claude Code?
10:30 AM
Create the skill file in ~/.claude/commands/ with YAML frontmatter (name + description). Then run /sync to push it to Cursor's skill folder. The sync command handles format differences automatically.

See wiki topic skills for the full workflow.
10:30 AM
What skills does Claude Code ship with that I haven't installed?
11:45 AM
Claude Code doesn't ship with installable skills — all your skills are custom. The Claude column in the skills workspace will show built-in slash commands once the Claude CLI integration is connected.

Right now only the Kimi integration is active.
