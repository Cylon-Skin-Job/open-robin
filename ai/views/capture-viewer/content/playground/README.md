# Playground

Isolated sandbox for visual experiments. HTML files here are **mockups**, not project code.

## Rules

- **Inline CSS only** — no external stylesheets, no build step
- **No JavaScript** — pure structure and style. If it needs JS, it's a prototype, not a playground file
- **Self-contained** — each `.html` file opens in a browser and looks complete
- **Not deployed** — nothing here ships. It's for exploring layout, color, spacing, typography

## Use Cases

- Mock up a modal before building the component
- Test a color scheme against real content
- Sketch a three-column layout for wiki navigation
- Design a dropdown or card grid in pure HTML/CSS
- Prototype how an evidence card would render visually
- Explore a knowledge graph node layout

## Naming

`{thing}-{variant}.html` — e.g., `wiki-sidebar-dark.html`, `evidence-card-compact.html`, `modal-confirm.html`
